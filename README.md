🚀 Discord Redirector (discord.kusokmedi.lat)Минималистичный и быстрый сервис для перенаправления пользователей на ваш Discord-сервер через кастомный домен. 🌐Демо: discord.kusokmedi.lat ✨📖 О проектеЭтот проект создан для тех, кто хочет иметь красивую, короткую и запоминающуюся ссылку для своего сообщества. Вместо длинных стандартных инвайтов Discord, вы можете использовать свой брендированный домен.✨ Основные преимущества:⚡ Мгновенный редирект: Скрипт срабатывает сразу после загрузки страницы.📱 Mobile Friendly: Корректно открывает приложение Discord на iOS и Android.🎨 SEO & OG Tags: Настроены мета-теги для красивого отображения превью ссылки в соцсетях и мессенджерах (Open Graph).🛠 Простота: Не требует базы данных или сложного бэкенда.🛠 ТехнологииHTML5 / CSS3 — базовая структура и стилизация.JavaScript — логика перенаправления.GitHub Actions / Vercel / Netlify — для автоматического деплоя (рекомендуется).🚀 Как запустить свой редиректорКлонируйте репозиторий:git clone [https://github.com/vash-logic/discord-redirect.git](https://github.com/vash-logic/discord-redirect.git)
cd discord-redirect
Настройте ссылку:Откройте файл index.html и замените значение переменной или ссылки в блоке <script>:const DISCORD_URL = "[https://discord.gg/ВАШ_КОД](https://discord.gg/ВАШ_КОД)";
Настройте мета-теги:Отредактируйте заголовки <meta> в index.html, чтобы при отправке ссылки отображалось название вашего сервера и описание.Деплой:Просто загрузите файлы на любой хостинг статических сайтов (GitHub Pages, Vercel или свой сервер).⚙️ Конфигурация Nginx (если используете свой сервер)Если вы хотите использовать чистый редирект на уровне сервера:server {
    listen 80;
    server_name discord.vash-domen.com;

    location / {
        return 301 [https://discord.gg/ВАШ_КОД](https://discord.gg/ВАШ_КОД);
    }
}
